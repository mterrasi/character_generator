Character Generator

  1. To build a character, click get started to generate the character's abilities
  2. Based on the abilities, complete the form to select the parameters for the character to be generated, or leave it set to random
  3. Click on generate character to send the parameters to the generator
  4. Optionally, download at JSON file to use with the interactive character sheet, made by Blacktower Games
  5. To reset the generator, click on generate new character
